# PixelMusicians

A Processing nevű Java alapú grafikus programozási nyelvben készült rövid 2D játék.

## Futtatás  

A futtatáshoz töltsd le a repositoryt, majd a Processing editor segítségével nyisd meg a gaem.pde filet, és futtasd azt.
